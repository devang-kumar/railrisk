# RailRisk AI

RailRisk AI is a disruption risk intelligence system. Rather than just predicting delays, it identifies which delays matter most, why they matter, and what actions should be taken first based on cargo criticality and downstream impacts.

## Project Structure

- `data/` - Contains simulated freight datasets for risk scoring.
- `src/` - Source code for risk scoring and prioritization (to be added).
- `docs/` - Documentation and project requirements (to be added).

## MVP Scope

Risk scoring and priority recommendation based on a simulated freight dataset with features including cargo type, destination, delay time, urgency level, and dependency type.
